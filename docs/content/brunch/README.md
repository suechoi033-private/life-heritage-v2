## 발행 규칙

- 모든 글은 **draft**. 실제 게시(외부 발송)는 **사장님이 직접**. (헌장 one-way door)
- **3분 분량**(공백 포함 1,200~1,500자). 상세는 본문 링크(지난 글·잇다 페이지)로.
- **브런치 글 = 인스타 카드뉴스 페어** — 원고 파일 하단에 6장 스펙 동봉, 같은 날 동시 발행.
- 매거진 2개: ①남은 사람의 일(에세이=구독 엔진) ②미리 알아두는 정리 노트(정보=검색 유입 엔진).
- 모든 글 끝 = **다음 글 예고 1줄 + 구독 유도 1줄** + **잇다 링크 1개(+UTM)**.
- 표 사용 금지(브런치 모바일 깨짐) — 문장 또는 이미지로.
- YMYL(재정·의료·법률): 수치·법률 단정 금지, "정부24·홈택스·전문가 확인" 안내 필수.
- ⚠️ 원고 내 "[지난 글](https://brunch.co.kr/@nonaksan)" 링크는 게시 시 **실제 글 URL로 교체** (사장님).
