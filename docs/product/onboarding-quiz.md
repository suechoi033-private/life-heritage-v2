# 잇다 본 가입 온보딩 진단 퀴즈 — 카피 확정안 (검토용)

> 작성: 2026-05-26. 마케팅(구조)+전략(매핑)+카피라이터(craft) 협업 산출.
> 상태: **카피 검토 단계.** 빌드·발행 전 창업자 승인 필요. 결정 대기 항목은 맨 아래.
> 정합 기준: welcome.html 어휘("자리"·"먼저 겪은 분"·도구 라우팅), ceremony-funnel.md(미러링·게이트·다크패턴 금지), CHARTER 가치4(YMYL).

## 진단 로직 (전략 확정 — 바꾸지 않음)
- stage는 **Q1·Q2 2축으로만** 판정(충돌 시 Q2 우선). 5 stage: 케어링/자기준비/회고/사별/탐색자 + `solo` 플래그(Q3=혼자).
- Q3~Q7은 stage를 **안 바꾸고** 결과 미러링·톤·CTA 강도에만 쓴다.
- stage별 첫 행동(activation): 케어링→케어일지+형제초대(care.html) · 자기준비→정보 한 가지(info.html) · 회고→오늘의 질문(ask.html) · 사별→먼저 겪은 분 글(forest.html) · 탐색자→콘텐츠 정독.

## 0. 여는 화면
- **헤드라인:** 지금, 당신은 어느 단계에 와 계신가요?  〔택1 → 결정 D-헤드라인〕
- **설명:** 일곱 번만 답하면, 지금 맞는 첫걸음을 같이 찾아요. 1분이면 돼요. 정답은 없어요.
- **버튼:** 시작하기
- **보조:** 답은 언제든 바꿀 수 있어요

## 1. Q1 (single) — 분기 시드
**지금 마음이 가장 많이 머무는 곳은 어디인가요?**
- 살아온 시간을 한번 돌아보고 싶어요 `reflect`
- 앞일을 미리 챙겨두고 싶어요 `pre`
- 곁에서 누군가를 돌보고 있어요 `care`
- 아끼던 분을 떠나보냈어요 `memory`  〔"소중한"↔"아끼던" 택1 → 결정 D-어휘1〕
- 아직 잘 모르겠어요 `explore`

## 2. Q2 (single) — 계기·trigger
**그 마음이 든 계기가 있다면, 어느 쪽에 가까운가요?**
- 부모님이나 가족이 편찮으세요
- 가까운 분을 얼마 전에 보내드렸어요
- 건강검진이나 진단에서 마음에 걸리는 게 있었어요
- 은퇴나 환갑, 손주처럼 삶의 매듭을 지났어요
- 특별한 계기는 없어요. 그냥 한번 챙겨보려고요

## 인터스티셜 A (판정된 stage 기준 — Q1 아님)
- 회고: "그러시군요. 살아온 시간을 한번 돌아보고 싶으신 거죠. 몇 가지만 더 여쭤볼게요."
- 자기준비: "그러시군요. 앞일을 미리 챙겨두고 싶으신 마음이군요. 몇 가지만 더 여쭤볼게요."
- 케어링: "그러시군요. 지금 누군가의 곁을 지키고 계시는군요. 몇 가지만 더 여쭤볼게요."
- 사별: "그러시군요. 아끼던 분을 떠나보내셨군요. 몇 가지만 더 여쭤볼게요."
- 탐색: "그러시군요. 아직 정하지 않으셔도 괜찮아요. 몇 가지만 더 여쭤볼게요."

## 3. Q3 (single) — 누구와 / solo 분기
**이 고민을 누구와 함께하고 계세요?**
- 형제, 자매와 함께요
- 배우자와 함께요
- 자녀와 함께요
- 지금은 혼자 하고 있어요 → `solo=true`

## 4. Q4 (single) — 마음의 무게 (톤 조절)
**지금 마음은 어느 쪽에 가까우세요?**
- 머릿속이 복잡하고 막막해요
- 잘 모르겠지만, 천천히 해보려 해요
- 가볍게, 한번 살펴보는 정도예요

## 5. Q5 (multi) — 마음에 걸리는 것
**아래 중 요즘 가장 많이 생각하는 것은 무엇인가요? 해당하는 것을 모두 골라주세요.**
- 내가 떠난 뒤 가족이 헤매지 않게 정리해 두는 것  〔"떠난 뒤"↔"언젠가" 택1 → 결정 D-어휘2〕
- 가족에게 못 한 말을 남기는 것
- 돌봄과 치료, 요양을 어떻게 할지
- 떠난 분을 잘 기억하고, 마음을 추스르는 것
- 내가 살아온 시간을 글로 남기는 것

## 6. Q6 (single) — 시작 강도 (CTA 형태)
**지금 시작한다면, 어느 정도가 편하시겠어요?**
- 짧은 글 하나 읽는 정도
- 질문 하나에 답해보는 정도
- 뭔가 하나 직접 정리해 보고 결과물을 보는 정도

## 7. Q7 (text, 선택) — 미러링 클라이맥스
**마지막으로, 사랑하는 사람을 떠올리며(본인이어도 괜찮아요) 지금 가장 남기고 싶은 한 문장이 있다면 적어주세요.**
- placeholder: "예: 고생한 우리 가족, 고맙다고 / 나는 이렇게 살았다고"
- 건너뛰기: "지금은 비워둘게요"

## 8. 결과 거울 (5분기 + solo)
각 분기: (a) 단계 명명 (b) 동행 한 줄 (c) 첫걸음.

**① 케어링 → care.html**
- (a) 당신은 지금, 누군가의 곁을 지키는 자리에 계세요.
- (b) 곁을 지키는 일은 혼자 짊어지기 버거워요. 나누면 한결 가벼워져요.  〔수치·"대부분" 없는 안전문. 인터뷰 후 실제 인용 교체 → 결정 D-출처〕
- (c) 오늘 면회 기록 한 줄을 남겨보세요. 형제도 같은 화면에서 함께 봅니다.

**② 자기준비 → info.html**
- (a) 당신은 지금, 가족이 헤매지 않게 미리 챙겨두려는 자리에 계세요.
- (b) 먼저 정리해 둔 분들은 '내 자식들은 나처럼 막막하지 않았으면' 하는 마음에서 시작했어요.
- (c) 가장 가벼운 것부터요. 가족에게 남기고 싶은 정보 한 가지를 적어보세요.  〔첫걸음을 "사전연명의료의향서"로 올릴지 = YMYL 감수 필요 → 결정 D-YMYL〕

**③ 회고 → ask.html**
- (a) 당신은 지금, 살아온 시간을 돌아보려는 자리에 계세요.
- (b) 거창하게 시작할 필요 없어요. 하루 한 문장이 일 년 뒤엔 한 권이 돼요.
- (c) 오늘의 질문 하나에 답해보세요. 1분이면 됩니다.

**④ 사별 → forest.html**
- (a) 당신은 지금, 아끼던 분을 떠나보낸 자리에 계세요.  〔어휘1 통일〕
- (b) 같은 길을 먼저 지난 분들의 기록이 여기 있어요. 천천히, 마음 가는 만큼만 읽으셔도 돼요.
- (c) 먼저 겪은 분의 이야기 한 편을 읽어보세요.  〔사별 직후는 도구 강제 X, Q4=막막이면 더 부드럽게〕

**⑤ 탐색자 → 콘텐츠 정독**
- (a) 아직 어디라고 정하지 않아도 괜찮아요. 천천히 둘러보는 것도 한 걸음이에요.
- (b) 많은 분들이 '그냥 한번' 들렀다가, 자기 자리를 찾았어요.  〔수치 없는 안전문 → 결정 D-출처〕
- (c) 마음 가는 글 한 편부터 읽어보세요.

**보조 · solo 오버레이 (위 분기 위에 덧댐)**
- (a 덧댐) 지금은 혼자 챙기고 계시지만, 그래서 더 또렷이 정리해 두는 분들이 있어요.
- (b) 혼자라서 더 막막한 게 아니라, 혼자라서 더 내 뜻대로 정리할 수 있어요.
- (c) 내 뜻을 한 가지 적어 두는 것부터요.

## 9. Q7 미러링
- **입력 있을 때** (단계 명명 바로 아래 인용블록):
  - 『지금 남기신 말 — "{입력}"』
  - 이 한 문장이, 지금 여기 와 계신 이유예요.
  - (자기준비 등에서는 CTA를 '가족에게 남기는 말'로 변주)
- **입력 없을 때:** 남기고 싶은 한 문장이 떠오르면, 언제든 적어두실 수 있어요.
- 원칙: 입력 문장은 **교정하지 않고 그대로** 보여준다.
