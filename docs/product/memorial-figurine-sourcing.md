## 배경 — 이 제품의 정체

- kaiven.shop / memofigurines.com / kaivenstore.com 은 모두 같은 제품을 파는 **드롭쉬핑 스토어**.
  실제 제작은 중국 공장의 풀컬러 3D 프린터(Sailner WJP, 호주산 레진, RoHS).
- 기술·공장은 이미 존재하고 누구나 접근 가능 → 진입장벽이 낮음.
  kaiven이 잘하는 건 제품이 아니라 "한 장의 사진 → 추모 피규어" 스토리 + 퍼널.
- 제작 흐름: 사진 → 3D 모델링(수작업 보정) → 풀컬러 레진 프린트 → UV 경화 → 핸드 피니싱. 보통 2~4주.

## ⚠️ 브랜드 톤 주의

- 제품(고인의 3D 추모 피규어 = 유품/추모품)은 잇다와 의미상 맞음.
- 하지만 **파는 방식은 베끼지 않는다.** kaiven식 `SAVE 41%` · `12k+ bought` · 별점 경쟁 · 긴급성은
  헌장(과시·경쟁 금지, 조용함·존엄)과 정면 충돌. "곁에 두는 작은 형태"라는 결로 간다.
- 추모(memorial)는 현재 보류·숨김 상태라는 점, 사별 맥락에선 품질 미달 시 정서적 타격이 크다는 점 유의.
- → 품질 통제가 사업 성패를 가른다.
