홈 (index.html)

기존 🕯 오늘 잇고 카드 1개의 목적지만 전용 지면으로 변경. 새 카드·새 자리 추가 없음 → 홈 복잡도 그대로.

콘텐츠 탭 (info.html)

라이프사이클 브라우저 상단에 "오늘 잇고" 입구 1줄 추가. 기존 주제 카테고리 구조와 충돌 없음.

전용 지면 (신설 1개)

오늘의 큐레이션 + 질문 + 지난 잇고 리스트. 고아였던 7편이 여기에 집을 가짐. 홈 피드의 reflection 제외 필터는 그대로 둠.

상세 (content-detail)

기존 페이지 재사용. DB·콘텐츠 그대로, 추가 작업 거의 0. 글 끝 CTA로 기록 도구 연결.
