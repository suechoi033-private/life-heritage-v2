# 終活ねっと(슈카츠넷, Shukatsu-net) 벤치마킹

> 조사일: 2026-05-19
> 한 줄 요약: **21세 도쿄대생이 SEO 콘텐츠로 2.5년 만에 月 1,000만 PV(월 1천만 페이지뷰)를 만들어 DMM에 약 10억 엔(약 100억 원)에 팔고, 3.5년 뒤 사업 자체가 사라진 회사. 성공의 청사진이자 실패의 해부도.**

> ※ 終活(슈카츠) = 인생 마지막을 준비하는 활동. "終わりの活動(끝의 활동)"의 약자. 한국어로는 "임종 준비" 또는 "사전 정리"에 해당.
> ※ YMYL = "Your Money Your Life" — 구글이 정의한 영역. 의료·법률·금융처럼 잘못된 정보가 사람 인생에 직접 피해 주는 콘텐츠. 검색 노출에서 가장 엄격하게 평가됨.

## 2. 콘텐츠 전략

### 카테고리 (전성기 2019년경)

| 일본어 원문 | 한글 읽기/뜻 | 비고 |
|---|---|---|
| お墓 | 오하카 (묘·납골당) | 가장 강력. 매칭 수익 최대 |
| 葬儀 | 소-기 (장례) | 핵심 SEO 타깃 |
| 相続 | 소-조쿠 (상속) | YMYL 위험 영역 |
| 介護 | 카이고 (개호 = 노인 돌봄) | |
| 保険 | 호켄 (보험 — 종신·의료) | |
| 仏壇·仏具 | 부츠단·부츠구 (불단·불구) | |
| 遺品整理 | 이힌세이리 (유품 정리) | |
| 終活全般 | 슈카츠 젠판 (종활 전반) | |
| 終活用語集 | 슈카츠 요-고슈 (종활 용어집) | SEO long-tail 입구 (`/word/` URL) |

### 콘텐츠 양·톤
- 정확한 기사 수 비공개. 추정: 누계 수천~1만 페이지
- **톤: 검색 의도 100% 정보형 가이드.** 「○○とは(○○란?)」「○○の費用(○○의 비용)」「○○の流れ(○○의 절차)」 형
- 에세이·체험기·인터뷰 **거의 없음** — 외부 비평가 지적 ("オリジナル(오리지널)·体験(타이켄 = 체험) 없다")
- SEO 도구: **MIERUCA(미에루카, Faber Company의 SEO 도구)** — 검색 의도 분석 + 표절 자동 체크 CMS 임베드

### 정보 아키텍처 (3계층)
- 用語集(요-고슈 = 용어집, `/word`) — long-tail 입구
- マガジン(매거진) 가이드 (`/article`) — 깊이 있는 가이드
- 比較表(히카쿠효 = 비교표, 매칭 LP) — 컨버전

## 3. 콘텐츠 생산 모델 (실패의 핵심)

| 항목 | 사실 |
|---|---|
| 핵심 생산자 | **학생 인턴 100명+** 동시 활동 (도쿄대·게이오·와세다·소피아 등) |
| 시급 | 1,100~1,700엔 (한화 약 1만 원~1만 5천 원), 월 60시간 이상 |
| 사전지식 | "終活(슈카츠) 지식 0에서 시작 OK" — 채용 페이지에 명문화 |
| 표절 방지 | **MIERUCA(미에루카)** 의 코피페 체크를 CMS에 자동 임베드 — **WELQ(웰크) 사태**에서 학습 |
| **의료/법률 監修(칸슈 = 감수)** | **공식 명단 없음.** YMYL 영역에 監修(감수)자 부재 = 시한폭탄 |
| 외부 비평 | "기사 내용이 정확하지 않다" 口コミ(쿠치코미 = 입소문/리뷰) 다수 |

### WELQ(웰크) 사태 — 배경
- **WELQ**: DeNA(디엔에이, 일본 IT 대기업)가 운영한 의료 정보 사이트
- 2016년 11~12월, 익명 라이터 양산 + 의사 監修(감수) 부재 + 표절 우회 매뉴얼 폭로 → **全(전) 기사 公開停止(코-카이 테이시 = 공개정지)** + 사회적 사건화
- 終活ねっと(슈카츠넷)는 WELQ 사태 직후 성장 → 표면(표절 체크 자동화)은 학습, 본질(전문가 監修)은 학습 못 함
- 결과: 행정 처분은 없었으나 **"신뢰를 천천히 잃는 슬로우 데스(slow death)"**

## 4. 비즈니스 모델

### 수익원
1. **葬儀仲介(소-기 츄-카이 = 장례 중개) 수수료** — 葬儀社(소-기샤 = 장례식장 회사) 매출의 **10~30%**
2. **霊園·墓地(레이엔·보치 = 공원묘지·묘지) 매칭 수수료**
3. **僧侶派遣(소-료 하켄 = 스님 파견) 수수료**
4. **광고 수익**

### 提携(테이케이 = 제휴) 파트너
- 葬儀社(장례식장) 약 **2,000사** (vs **鎌倉新書(카마쿠라신쇼)** 의 「いい葬儀(이이 소-기 = 좋은 장례)」 5,000사+ — 2.5배 격차)
- 霊園·墓地(공원묘지) 약 5,000건
- 僧侶(스님) 1,000명 규모

### 가격
- 「DMMのお葬式(DMM의 장례)」 정액 플랜: **145,000엔(税込 = 제이코미 = 세금 포함, 약 145만 원)부터 7개 플랜**

### 매출 / 인수가
- 終活ねっと(슈카츠넷) 단독 결산 비공개
- TechCrunch JP 보도: DMM이 51% 인수 시 **약 10억 엔(약 100억 원) 추정**, 잔여 49% 가격은 비공개
- 시장 참고: 일본 葬儀(장례) 시장 약 **1.5~1.8조 엔 (약 15~18조 원)**

## 6. UI/UX

- 다층 구조: マガジン(매거진, syukatsulabo.jp) + 매칭/검색 + 정액 플랜 LP
- **퍼널**: SEO 진입 → 가이드 기사 → 比較表(히카쿠효 = 비교표)/CTA → 相談(소-단 = 상담) 폼 → 콜백 → 매칭 → 葬儀社(장례식장) 성약
- 회원제 가치 약함 (transaction 중심)
- 2021 리뉴얼 공식 표현: "見やすい(미야스이 = 보기 쉬운)·わかりやすい(와카리야스이 = 이해하기 쉬운)·調べやすい(시라베야스이 = 찾기 쉬운)" → **정보량 과다 자인**

## 8. 흥망성쇠 — 잇다가 외워야 할 5가지 실패 원인

### 1. 광고 단가 폭등
- 2019→2022 Google·SNS 광고 CPC(클릭당 단가) **약 1.5배 상승**
- SEO + paid(유료 광고) 둘 다 비싸지자 매칭 마진 잠식
- j-cast(니혼 IT 매체) 사후 보도가 撤退(텟타이 = 철수) 직접 원인으로 지목

### 2. YMYL 양산 모델의 시한폭탄
- 학생 인턴 100명, 監修(감수) 없음
- 행정 처분은 없었지만 평판 누적 악화
- Google의 Helpful Content Update(2022) 같은 알고리즘 변동에 약함

### 3. 매칭 단일 수익의 모순
- 葬儀社(장례식장)가 매출의 10~30%를 수수료로 떼임 → 業界(교-카이 = 업계) 반발
- 業界(업계) 비평가 지적: *"수수료 부담이 葬儀社(장례식장)의 ブラック化(부락쿠카 = 악덕업체화)를 가속, 紹介媒体(쇼-카이 바이타이 = 소개 매체)이 業界를 갉아먹는 構造(코-조 = 구조)"*
- 코로나로 **家族葬(카조쿠소 = 가족장)·直葬(쵸쿠소 = 직접화장)** 비중 증가 → 1건당 紹介料(쇼-카이료 = 소개료) 단가 하락

### 4. 콘텐츠 ↔ 매칭 LP 분리의 결정타
- 2021년 10월 리뉴얼에서 「DMMのお葬式(장례)/お坊さん(스님)/お墓探し(묘 찾기)」 **3개로 분리**
- SEO 도메인 권위(도메인 어소리티) 분산
- 7개월 후 撤退(철수) 발표

### 5. 인수 후 ramp-up 강박
- 타임라인: DMM 인수(2018.10) → TV CM(2019.12) → 완전자회사화(2020.03) → 全面(젠멘 = 전면) 리뉴얼(2021.10) → 撤退(철수, 2022.05)
- 모회사 손해처리 압력 + 시너지 강박 = 사업 왜곡
- 창업자 退任(퇴임) 후 brand 방향 잃음

## 9. 살아남은 경쟁자 비교

| 회사 | 모델 | 차별화 | 현 위치 |
|---|---|---|---|
| **鎌倉新書(카마쿠라신쇼)** (1984~) | 매칭 + 출판 + B2B SaaS + 보험·相続(상속) | **30년 業界(업계) 신뢰**, 提携(제휴) 5,000사+, **사업 다각화** | **東証プライム(도-쇼 프라임 = 도쿄증권거래소 프라임) 상장**, 매출 30억 엔+ (약 300억 원), CAGR(연평균 성장률) 26% |
| **よりそう(요리소우)** (2009~) | 매칭 + 「お坊さん便(오보-산빈 = 스님 출장 서비스)」 + 메이저 VC | 정액제 혁신, 終活(종활) 통합 회원제 | 2019년 20억 엔(약 200억 원) 조달, 강한 SEO/브랜딩 |
| **小さなお葬式(치이사나 오소-시키 = 작은 장례)** (운영사: **ユニクエスト(유니퀘스트)**) | 정액제 자체 시공 | 단순 가격 + 강력한 광고 | 매칭 1위급 |
| **終活ねっと(슈카츠넷)** | SEO + 매칭만 | 콘텐츠 양·SEO만 | **소멸** |

→ **한 줄: 콘텐츠 + 매칭 단일 트랙은 망한다. 콘텐츠 + 매칭 + B2B + 출판 + 다각화 = 산다.**

## 10. 잇다에 적용할 lessons

### 베껴야 할 것
1. **用語集(용어집) + 가이드 + 체크리스트 3계층 정보 아키텍처**
2. **MIERUCA(미에루카)식 표절 자동검수 CMS** — 한국에서도 사고 1건이 신뢰 무너뜨림
3. **창업자 스토리텔링** — "21세 도쿄대생"같이 자산화
4. **SEO 입구 전략 그 자체는 옳다** — 2.5년에 月 1,000만 PV는 합리적 곡선

### 절대 피해야 할 것
5. **YMYL 양산 = 시한폭탄** → 변호사·세무사·장례지도사·의사·심리사 監修(감수) 명단 visible
6. **매칭 단일 수익 금지** → 시작부터 다리 4개 (매칭 + 회원제 + 도구 유료화 + B2B SaaS)
7. **콘텐츠와 도구를 같은 도메인·앱에서 끝까지 운영**
8. **VC 받고 paid ramp 금지** — organic(자연 유입) 안정될 때까지 paid(유료 광고) 최소
9. **"정보형 가이드만" = 사람 안 머문다** → 체험기·인터뷰·에세이·커뮤니티 필수

### 한국 vs 일본 차이
10. **검색 다채널화**: 한국은 Naver(네이버) + Kakao(카카오) + YouTube + Instagram **동시 필수**. 일본은 Google SEO 1축으로 충분했음
11. **타깃 2개**: (a) 본인 준비형 (일본 終活의 메인) + (b) 4050 자녀 돌봄형 (한국 실정) **동시 타게팅**

### 잇다가 終活ねっと(슈카츠넷)보다 잘할 수 있는 것
12. **App-native lifecycle 도구** — 슈카츠넷은 web/매칭폼만, 잇다는 푸시·매년 업데이트·가족 공유
13. **AI 활용** — 1차 콘텐츠 초안 + 감수자 sign-off 워크플로우 (단 YMYL이므로 감수 필수)
14. **커뮤니티** — 終活ねっと(슈카츠넷)의 가장 큰 약점
