# 終活ねっと (Shukatsu-net) 벤치마킹

> 조사일: 2026-05-19
> 한 줄 요약: **21세 도쿄대생이 SEO 콘텐츠로 2.5년 만에 月 1,000만 PV 만들어 DMM에 약 10억엔에 팔고, 3.5년 뒤 사업 자체가 사라진 회사. 성공의 청사진이자 실패의 해부도.**

## 2. 콘텐츠 전략

### 카테고리 (전성기)
- お墓 (묘지·영원) — 가장 강력, 매칭 수익 최대
- 葬儀 (장례) — 핵심 SEO 타겟
- 相続 (상속) — YMYL 위험
- 介護 (개호·돌봄)
- 保険 (종신·의료보험)
- 仏壇·仏具
- 遺品整理
- 終活全般 + 終活用語集 (`/word/` URL — 용어집 SEO 입구)

### 콘텐츠 양·톤
- 정확한 기사 수 비공개. 추정: 누계 수천~1만 페이지
- **톤: 검색 의도 100% 정보형 가이드**. 「○○とは」「○○の費用」「○○の流れ」
- 에세이·체험기·인터뷰 **거의 없음** — 외부 비평가 지적 ("オリジナル·体験 없다")
- SEO 도구: **MIERUCA (Faber Company)** — 검색 의도 분석 + 표절 자동 체크 CMS 임베드

### 정보 아키텍처
3계층:
- 용어집 (`/word`) — long-tail 입구
- 매가진 가이드 (`/article`) — 깊이 있는 가이드
- 비교표 (매칭 LP) — 컨버전

## 3. 콘텐츠 생산 모델 (실패의 핵심)

| 항목 | 사실 |
|---|---|
| 핵심 생산자 | **학생 인턴 100명+ 동시 활동** (도쿄대·게이오·와세다 등) |
| 시급 | 1,100~1,700엔, 월 60시간 |
| 사전지식 | "終活 지식 0에서 시작 OK" — 명문화 |
| 표절 방지 | MIERUCA 코피페 체크 자동 (WELQ식 사고 방지) |
| **의료/법률 監修** | **공식 명단 없음. YMYL 영역에 監修 부재가 시한폭탄** |
| 외부 비평 | "기사 내용이 정확하지 않다" 口コミ 다수 |

### WELQ 사태와의 관계
- WELQ(DeNA 의료): 2016.11~12 全 기사 公開停止 (익명 양산·無 監修)
- 終活ねっと는 WELQ 직후 성장. **표면적으로는 학습 (코피페 체크 도입), 본질은 학습 못함 (전문가 監修 부재)**
- 행정 처분은 없었으나 "신뢰를 천천히 잃는 슬로우 데스"

## 4. 비즈니스 모델

### 수익원
1. **葬儀仲介 수수료** — 葬儀社 매출의 **10~30%**
2. **霊園·墓地 매칭 수수료**
3. **僧侶 派遣 수수료**
4. **광고 수익**

### 提携 파트너
- 葬儀社 약 2,000건 (vs 鎌倉新書「いい葬儀」5,000건 — 2.5배 격차)
- 霊園·墓地 약 5,000건
- 1,000명 규모 僧侶 네트워크

### 가격
- 「DMMのお葬式」 정액 플랜: 145,000엔(税込)부터 7개 플랜

### 매출
- 終活ねっと 단독 결산 비공개
- 인수가: TechCrunch JP "약 10억엔(51% 분) 추정", 잔여 49% 비공개
- 시장 참고: 일본 葬儀 시장 약 1.5~1.8조엔

## 6. UI/UX

- 다층 구조: 매가진(syukatsulabo.jp) + 매칭/검색 + 정액 플랜 LP
- **퍼널**: SEO 진입 → 가이드 기사 → 비교표/CTA → 상담 폼 → 콜백 → 매칭 → 葬儀社 성약
- 회원제 가치 약함 (transaction 중심)
- 2021 리뉴얼 공식 표현: "見やすい·わかりやすい·調べやすい" → **정보량 과다 자인**

## 7. 마케팅 / Acquisition

## 8. 흥망성쇠 — 잇다가 외워야 할 5가지 실패 원인

### 8-1. 광고 단가 폭등
- 2019→2022 Google·SNS CPC 약 1.5배
- SEO + paid 둘 다 비싸지자 매칭 마진 잠식
- j-cast 사후 보도가 撤退 직접 원인으로 지목

### 8-2. YMYL 양산 모델의 시한폭탄
- 학생 인턴 100명, 監修 없음
- 행정 처분은 없었지만 평판 누적 악화
- Helpful Content Update 등 알고리즘 변동에 약함

### 8-3. 매칭 단일 수익의 모순
- 葬儀社가 매출의 10~30%를 수수료로 떼임 → 業界 반발
- "수수료 부담이 葬儀社의 ブラック化 가속, 紹介媒体이 業界를 갉아먹는 構造" (考える葬儀屋さん)
- 코로나로 家族葬·直葬 비중 증가 → 1건당 紹介料 단가 하락

### 8-4. 콘텐츠 ↔ 매칭 LP 분리의 실수 (마지막 일격)
- 2021.10 「DMMのお葬式 / お坊さん / お墓探し」 3개 분리
- SEO 도메인 권위 분산
- 7개월 후 撤退 (2022.05)

### 8-5. 인수 후 ramp-up 강박
- DMM 인수(2018.10) → TV CM(2019.12) → 완전자회사화(2020.03) → 全面 리뉴얼(2021.10) → 撤退(2022.05)
- 모회사 손해처리 압력 + 시너지 강박 = 사업 왜곡
- 창업자 退任 후 brand 방향 잃음

## 9. 살아남은 경쟁자 비교

| 회사 | 모델 | 차별화 | 현 위치 |
|---|---|---|---|
| **鎌倉新書** (1984~) | 매칭 + 출판 + B2B SaaS + 보험 | 30년 業界 신뢰, 提携 5,000+, **사업 다각화** | **東証프라임 上場**, 매출 30억엔+, CAGR 26% |
| **よりそう** (2009~) | 매칭 + お坊さん便 + 메이저 VC | 정액제 혁신, 終活 통합 회원제 | 2019년 20억엔 조달, 강한 SEO/branding |
| **小さなお葬式** (ユニクエスト) | 정액제 자체 시공 | 단순 가격 + 강력한 광고 | 매칭 1위급 |
| **終活ねっと** | SEO + 매칭만 | 콘텐츠 양·SEO만 | **소멸** |

→ **한 줄: 콘텐츠 + 매칭 단일 트랙은 망한다. 콘텐츠 + 매칭 + B2B + 출판 + 다각화가 있어야 산다.**

## 10. 잇다에 적용할 lessons

### 베껴야 할 것
1. **용어집 + 가이드 + 체크리스트 3계층 정보 아키텍처**
2. **MIERUCA식 표절 자동검수 CMS** — 한국에서도 사고 1건이 신뢰 무너뜨림
3. **창업자 스토리텔링** — "21세 도쿄대생" 같은 자산화
4. **SEO 입구 전략 그 자체는 옳다** — 2.5년에 月 1,000만 PV는 합리적 곡선

### 절대 피해야 할 것
5. **YMYL 양산 = 시한폭탄** → 변호사·세무사·장례지도사·의사·심리사 監修 명단 visible
6. **매칭 단일 수익 금지** → 시작부터 다리 4개 (매칭 + 회원제 + 도구 유료화 + B2B SaaS)
7. **콘텐츠와 도구를 같은 도메인·앱에서 끝까지 운영**
8. **VC 후 paid ramp 금지** — organic 안정될 때까지 paid 최소
9. **"정보형 가이드만" = 사람 안 머문다** → 체험기·인터뷰·에세이·커뮤니티 필수

### 한국 vs 일본 차이
10. **검색 다채널화**: 한국은 Naver + Kakao + YouTube + Instagram 동시 필수
11. **타깃 2개**: (a) 본인 준비 (b) 4050 자녀 돌봄 동시 타게팅

### 잇다가 더 잘할 수 있는 것
12. **App-native lifecycle 도구** — 終活ねっと는 web/매칭만
13. **AI 활용** — 1차 초안 + 監修자 sign-off 워크플로우
14. **커뮤니티** — 終活ねっと 약점
