## 2. 레몬테라스 신호 — 결정 근거 요약

### 카페 정체
- **회원 3,016,256명** · 글 14,878,806개 · 22년 된 숲등급 메가카페
- **여성 전용 / 1966~2006년생 (20~60세)** = 잇다 케어링·유언 두 path 타겟과 정확 일치
- 결혼/새댁 → 워킹맘 → 학부모/살림 → 시니어 케어가 한 카페에 다 있음

### "유언" 검색 결과 4가지 결 (사장님 캡쳐 분석)

| 결 | 대표 글 (조회·댓글) | 의미 → 잇다 자리 |
| :-- | :-- | :-- |
| **A. 실무 질문형** | 유언공증 증인(176·0) · 유언 신탁 수수료(123·0) · 변호사 비용(240·1) | **답이 거의 없음** → 잇다 도구·SEO 글이 답할 자리 |
| **B. 부모 케어 + 유언** | "할머니가 며칠 못 산다 하시는데… 유언 같은 걸 남겨요?"(373·10) · "유언은 꼭 들어드려야 할까요"(1,045·14) | **두 path 교집합** — 잇다 핵심 페르소나 |
| **C. 본인 유언 의향형** | "레테님들은 나중에 죽으면 어떻게 해달라고 유언하고 싶으세요?"(722·23) | **유언/웰니스 path 정확 명중** |
| **D. 뉴스/가십** | 故 오요안나 유서, 김수미 유언곡 | 패스 |

### 결정에 닿는 3가지 신호

1. **(b) 강한 근거** — 같은 카페 같은 사람이 "유언"도 "부모 케어"도 검색. 분리하면 같은 사람을 두 번 끌어와야 함, 비효율.
2. **댓글이 거의 없음** — 변호사 1명("정진 변호사")이 비공식 답하는 문화. 잇다 운영자 답글이 자연스럽게 들어갈 자리.
3. **망고하다 부재** — 키워드도 운영자 답글도 보이지 않음. 디지털 모집단을 못 잡고 있음 = 사장님 판단 적중.

## 3. 망고하다 정리 (벤치마크 종결)

- **위상**: 카테고리 존재 증명자 (경쟁자 X)
- **무엇을 했나**: 한국에서 "유언/웰다잉" 카테고리 자체가 가능하다는 증명 + 변호사 자문·특허 유언 템플릿 · 잘 죽기 위해 잘 살기 철학
- **왜 잇다 경쟁자가 아닌가**: ① 오프라인 위주(서비스 도구화 약함) ② 유저 규모 미미 ③ 1인 유언 중심(가족 협업 0) ④ 무거운 진입(변호사·법적 효력 강조)
- **잇다와 다른 자리**:
  - 망고하다: 법률 자동화 / 1인 유언 / 청년 well-dying 캠페인
  - **잇다**: 관계 자동화 / **가족 협업 유언** / 케어링 ↔ 유언 자연 전환 / 다세대(20~60대)
- **벤치마크 문서 후속**: `docs/benchmark/korean-apps-2026-06-15.md` 망고하다 칸은 "카테고리 검증자" 한 줄로 마무리. 사장님이 나중에 유언 작성해 보면 가입 흐름만 메모.

### 운영자 답글 톤 (헌장 일관)
- 광고 X / 정보 O. "이런 도구도 있어요" 식 1줄 + 링크만.
- 변호사·법적 효력 단정 X. YMYL은 "공증사무소·변호사 상담 권유"로 한 줄 안내.
- 카페 답글 → 잇다 클릭한 사람에게 잇다 진입 화면은 카페 검색어와 동일한 카드(유언/케어링) 우선.
