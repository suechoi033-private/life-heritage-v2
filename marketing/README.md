1. **첫 3초 = 한 사람의 한 마디.** 추상명제 금지.
2. **1인칭 고백 톤.** 훈계·영업·종교색 금지.
3. **'미루다 놓친 후회' 서사.** 잇다 슬로건과 정합.
4. **끝은 한 가지 작은 행동.** "전화 한 통" / "한 줄 메모".
5. **잇다 = 그 행동의 도구.** 서비스 직판 멘트 금지, 자연스러운 안내.
